# Ambiente de Desenvolvimento

## Requisitos

- Python 3.11+
- Node 22+
- Git
- VS Code
- Android Studio

## Instalação

### Python

```bash
python -m venv venv
```

### Ativar

Windows

```bash
venv\Scripts\activate
```

Linux

```bash
source venv/bin/activate
```

### Dependências

```bash
pip install -r backend/requirements-dev.txt
```

```bash
npm install
```

## Executar

```bash
python backend/app.py
```

## Mobile

```bash
npm run sync

npm run android
```
